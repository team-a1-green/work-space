# 🎯 おみくじシステム


HTML・CSS・Bootstrap・JavaScriptを使った、「おみくじWebアプリ」です。  
運勢・コメント・ラッキーアイテム・ラッキーな場所・ラッキーな言葉をランダムに表示します。
